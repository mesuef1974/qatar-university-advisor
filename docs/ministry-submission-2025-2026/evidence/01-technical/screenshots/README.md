# لقطات الشاشة — الأدلة التقنية

## ملاحظة هامة حول اللغة والاتجاه

> **التطبيق عربي أولاً** — لا يدعم i18n إنجليزي فعلياً حتى الآن.
>
> اللقطات المسمّاة `ltr-*` تُظهر اتجاه LTR للمقارنة البصرية فقط، لكن **المحتوى يبقى عربياً**.
> النسخة الإنجليزية الكاملة (ترجمة + localization شاملة) مخطط لها في خارطة الطريق ولم تُنفّذ بعد.
>
> السبب في إدراج لقطات LTR هو توضيح أن تصميم الواجهة **مرن بصرياً** (grid + spacing + components) ويحتمل مستقبلاً اتجاهاً أخر دون كسر الـ layout — لكنها ليست دليلاً على دعم لغة إنجليزية فعلي اليوم.
