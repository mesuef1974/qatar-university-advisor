## قواعد التوثيق الإلزامية

| المستوى | التوثيق المطلوب | الموعد | المسؤول |
|---------|----------------|--------|---------|
| **قبل التنفيذ** | PRD أو وصف المهمة | قبل بدء الكود | elite-product / elite-engineering |
| **أثناء التنفيذ** | Commit messages واضحة (Conventional Commits) | مع كل commit | المطور المنفّذ |
| **بعد التنفيذ** | تحديث الشيك ليست + سجل التغييرات | فور الانتهاء | المسؤول عن المهمة |
| **عند الأخطاء** | تسجيل في سجل الثغرات أو الحوادث | خلال ساعة | elite-devops / elite-qa |
| **عند القرارات** | ملف مستقل في `08_القرارات-الإدارية/` | في يوم القرار | elite-secretariat |
| **عند الإصدارات** | تحديث CHANGELOG + git tag | مع كل release | elite-devops |
