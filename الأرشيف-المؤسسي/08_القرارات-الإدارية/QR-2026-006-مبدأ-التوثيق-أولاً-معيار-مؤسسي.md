## نص القرار الرسمي

## قواعد التوثيق الإلزامية

| المستوى | التوثيق المطلوب | الموعد | المسؤول |
|---------|----------------|--------|---------|
| **قبل التنفيذ** | PRD أو وصف المهمة | قبل بدء الكود | azkia-product / azkia-engineering |
| **أثناء التنفيذ** | Commit messages واضحة (Conventional Commits) | مع كل commit | المطور المنفّذ |
| **بعد التنفيذ** | تحديث الشيك ليست + سجل التغييرات | فور الانتهاء | المسؤول عن المهمة |
| **عند الأخطاء** | تسجيل في سجل الثغرات أو الحوادث | خلال ساعة | azkia-devops / azkia-qa |
| **عند القرارات** | ملف مستقل في `08_القرارات-الإدارية/` | في يوم القرار | azkia-secretariat |
| **عند الإصدارات** | تحديث CHANGELOG + git tag | مع كل release | azkia-devops |
