## 🗺️ Conversation State Machine (مُعتمد من CEO)

```
┌─────────────────────────────────────────────────────────┐
│              نموذج دورة حياة المحادثة                    │
│                                                         │
│  STAGE_0: هوية        → "أنت طالب أم ولي أمر؟"         │
│      ↓                                                  │
│  STAGE_1: خلفية       → "جنسية + مسار + معدل"          │
│      ↓                                                  │
│  STAGE_2: ميول        → "ما يشغل تفكيرك؟"             │
│      ↓                                                  │
│  STAGE_3: تخصصات      → قائمة مخصصة + سوق العمل        │
│      ↓                                                  │
│  STAGE_4: جامعات      → مطابقة معدل + جنسية + تخصص     │
│      ↓                                                  │
│  STAGE_5: تمويل       → منح + سبونسرز                  │
│      ↓                                                  │
│  STAGE_6: ملخص        → تقرير شخصي كامل                │
│                                                         │
│  * الانتقال ليس خطياً — قد يقفز المستخدم بين المراحل    │
│  * كل مرحلة تُحدَّث عند الحصول على معلومات جديدة        │
└─────────────────────────────────────────────────────────┘
```
