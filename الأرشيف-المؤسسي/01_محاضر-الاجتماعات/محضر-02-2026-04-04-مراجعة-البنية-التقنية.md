### T-007 — إصلاح خلل HMAC-SHA256 الحرج ✅

**الخلل:** `api/webhook.js` كان يستخدم `JSON.stringify(req.body)` لحساب HMAC بدلاً من الـ raw bytes الأصلية من Meta.

**النتيجة:** رفض 403 لجميع الرسائل الواردة من WhatsApp.

**الإصلاح المُنفَّذ:**
```javascript
// إيقاف Vercel bodyParser
export const config = { api: { bodyParser: false } };

// قراءة الـ raw bytes
async function getRawBody(req) {
  const chunks = [];
  req.on('data', chunk => chunks.push(Buffer.isBuffer(chunk) ? chunk : Buffer.from(chunk)));
  return new Promise((resolve) => req.on('end', () => resolve(Buffer.concat(chunks))));
}

// التحقق من الـ HMAC بالـ raw string
const rawBodyBuffer = await getRawBody(req);
const rawBodyStr = rawBodyBuffer.toString('utf8');
if (!verifySignature(rawBodyStr, signature)) { ... }
body = JSON.parse(rawBodyStr); // اليدوي بعد التحقق
```

**نتيجة الاختبار:** ✅ POST 200 - رسائل تُعالج بنجاح

### T-003 — Knowledge Base Manager ✅

**الملف:** `lib/knowledge-base.js` — 250+ سطر

**الوظائف:**
```
normalizeText()        — تطبيع النص العربي (حركات/همزات/ياء)
jaccardSimilarity()    — حساب نسبة التشابه (0.0 - 1.0)
detectCategory()       — تصنيف السؤال تلقائياً
searchKnowledgeCache() — البحث بحد أدنى 40% تشابه
saveToKnowledgeCache() — حفظ ردود Gemini تلقائياً
incrementCacheUse()    — رفع quality_score تدريجياً
logAnalytics()         — تسجيل الإحصاءات
getFromKnowledgeBase() — الدالة الرئيسية
```

**الخوارزمية:**
1. تطبيع النص (إزالة تشكيل، توحيد حروف)
2. كشف الفئة (universities/scholarships/salary...)
3. استرجاع أفضل 50 إجابة من نفس الفئة
4. حساب Jaccard Similarity لكل إجابة
5. Weighted Score = similarity × (0.7 + quality_score × 0.3)
6. إعادة أفضل نتيجة إذا ≥ 40%
