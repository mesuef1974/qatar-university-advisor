### SEC-001
**Timing Attack على كلمة مرور Admin**

| الحقل | القيمة |
|---|---|
| **المعرّف** | SEC-001 |
| **النوع** | Security |
| **الشدة** | 🔴 CRITICAL |
| **الأولوية** | P0 — إصلاح فوري |
| **الوصف** | مقارنة كلمة مرور Admin باستخدام عامل `!==` العادي بدلاً من مقارنة cryptographic آمنة |
| **الملف والسطر** | `api/admin.js:39` |
| **التأثير** | Timing Attack — يمكن لمهاجم قياس زمن استجابة الـ API وتخمين كلمة المرور حرفاً بحرف مما يُمكّنه من اختراق لوحة الإدارة بالكامل |
| **السبب الجذري** | المقارنة النصية العادية (`===` / `!==`) تستغرق وقتاً متفاوتاً بحسب عدد الأحرف المتطابقة، مما يُكشف معلومات عن كلمة المرور عبر قناة جانبية زمنية |
| **الحل المطبّق** | `crypto.timingSafeEqual()` مع Buffer comparison — يضمن وقت معالجة ثابتاً بغض النظر عن صحة المدخل |
| **المكتشف بواسطة** | فريق الأمن السيبراني — Sprint طارئ 07 |
| **تاريخ الاكتشاف** | 2026-04-04 |
| **تاريخ الإصلاح** | 2026-04-04 |
| **الحالة** | ✅ مغلق |
| **Sprint** | Sprint طارئ 07 |

**ملاحظة تقنية:** يجب اختبار أي endpoint يقبل كلمة مرور بـ timing analysis للتأكد من استخدام `timingSafeEqual` أو ما يعادله في كل مكان.

```
┌─────────┐     ┌─────────────┐     ┌─────────┐     ┌──────────┐     ┌────────┐
│  Open   │────▶│ In Progress │────▶│  Fixed  │────▶│ Verified │────▶│ Closed │
└─────────┘     └─────────────┘     └─────────┘     └──────────┘     └────────┘
    │                                                                       ▲
    │                         ┌──────────┐                                 │
    └────────────────────────▶│ Rejected │─────────────────────────────────┘
                              └──────────┘
                          (ليست مشكلة أو مكررة)
```

| المرحلة | الوصف | المسؤول |
|---|---|---|
| **Open** | المشكلة مُسجَّلة وتنتظر التعيين | مكتشف المشكلة → مدير QA |
| **In Progress** | جارٍ التحقيق والإصلاح | مدير القسم المسؤول |
| **Fixed** | الإصلاح مُطبَّق وجاهز للاختبار | المهندس المسؤول |
| **Verified** | تم التحقق من الإصلاح بواسطة QA | مدير QA |
| **Closed** | مُغلَق رسمياً مع التوثيق الكامل | مدير QA + الأرشيف |
| **Rejected** | ليست مشكلة أو مكررة أو خارج النطاق | مدير QA + صاحب المشكلة |
