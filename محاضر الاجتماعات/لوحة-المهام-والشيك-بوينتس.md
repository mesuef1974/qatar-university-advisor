**المشكلة:** "منح غير القطريين" كانت تُرجع scholarship_amiri بدلاً من scholarship_non_qatari

**جذر المشكلة (3 أخطاء):**
1. `q.includes('غير قطري')` لا يتطابق مع "غير القطريين" (بسبب "ال" التعريف)

3. لا يوجد فحص جنسية الملف الشخصي لتصحيح النتيجة

**الإصلاحات:**
- [x] إضافة صيغ عربية شاملة: `غير القطريين`, `للمقيمين`, `وافدين`, `للوافدين`, `أجانب`, `للأجانب`, `لغير القطريين`
- [x] إعادة ترتيب catch-all المنح: غير القطريين أولاً ثم القطريين
- [x] تصحيح تلقائي في `processMessage()` ← profile.nationality = non_qatari → يُحوّل scholarship_amiri إلى scholarship_non_qatari
