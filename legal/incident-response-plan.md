# خطة الاستجابة لاختراق البيانات
## Incident Response Plan — PDPPL Art. 16

## مستويات الاختراق:
- **Level 1**: وصول غير مصرح به (منخفض الخطورة)
- **Level 2**: تسريب بيانات محدود
- **Level 3**: تسريب بيانات واسع (يستوجب إبلاغ الجهات)

## الإجراءات الفورية (خلال 72 ساعة):
1. عزل النظام المتأثر
2. توثيق الحادثة
3. إبلاغ المركز الوطني للأمن الإلكتروني (NCSA) — Level 3 فقط
4. إشعار المستخدمين المتأثرين
